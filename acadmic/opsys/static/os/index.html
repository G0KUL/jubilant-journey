Introduction

Operating systems are integral part of any electronic device. All electronic devices have certain way of operation. They have some standard operation they can perform.
We consider the example of a washing machine. In a washing machine we have certain operations we can perform. Let 2 operations be washing the cloth and drying the cloth. Now when washing machine is switched on, it has certain things it does before you start to press any button in it. This is normally the time taken to boot the system.
