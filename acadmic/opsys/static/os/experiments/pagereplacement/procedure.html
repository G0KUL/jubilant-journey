Procedure

Simulation covers the common page replacement algorithms like Optimal Page Replacement, Least Recently Used and First In First Out.

Procedure is as follows:
  1. Open the Simulation page
  2. Student has to provide 3 inputs values
  3. Enter the reference string. Reference string is the sequence of pages requested from the secondary memory.
    • Page number must be single character.
    • You can use the characters [a-z, A-Z, 0-9] to represent a page number.
    • Don't use any separator between page number.
    • ex. if reference string is [1,2,3,4,0,a,b,C] then you should provide input as: "12340abC" without double quotes.
  4. Enter the number of frames available in main memory.
  5. Choose appropriate algorithm for the simulation.
  6. Click Simulate Button to see the Visual Simulation of Page Replacement step by step
  7. Click next button to see the next step and observe the changes
  8. Click Next button every time to perform next step
