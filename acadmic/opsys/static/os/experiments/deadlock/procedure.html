Procedure

  • In Dining philosopher problem count of number of philosophers is always odd and generally number of philosophers are >= 5.
  • Each philosopher is associated with two actions, either thinking or hungry
  • At a time philosopher can be either thingking of hungry.
  • These actions can simulated using hungry and thinking button
  • Initially all the philosophers are in the thinking state.
  • If philosopher is hungry then click the hungry button. If both the chopsticks (on left right) are available then philosopher starts eating else he/she waits for chopsticks to become available
  • If all the philosophers are hungry then at a time only two of them can eat and others has to wait
  • Solution is deadlock free
