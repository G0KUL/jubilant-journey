Procedure

This simulation covers different scheduling algorithms like First Come First Serve, Shortest Job First, Shortest Remaining Time First, Round Robin and Priority scheduling. Student can solve one scheduling example using different scheduling algorithms and can find out the difference in the performance.

In order to run this simulation, student has to select the desired algorithm first and give necessary input values. Following is the detailed procedure to perform the experiments:

  1. Open Simulation page
  2. Choose the scheduling algorithm of your interest.
  3. Enter the count of participating processes.
  4. Enter the input values:
    1. If the algorithm selected is FCFS or SJF or SRTF then
      • Enter the Arrival Time and Burst Time for each process
    2. If the algorithm selected is Round Robin then
      • Enter the Burst time of each process
      • Enter the Quantum time
    3. If the algorithm selected is Priority Based scheduling which is non-preemptive version:
      • Enter the Priority, Arrival Time and Burst time for every process
      • Bigger number is higher priority
  5. Click on Simulate button
  6. Simulation begins, the table below 'Next' button will show the order in which processes are executed and in main table student can see the Finish time, Turn Around Time and Waiting Time calculated for each process whose execution is completed.
  7. Keep clicking next button till all the processes are not executed.
  8. Simulation ends once execution of all processes is over.
